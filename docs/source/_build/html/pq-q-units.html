Using PhysicalQuantites in Python¶

%precision 2
from PhysicalQuantities import q

The convenience class ‘q’ can be used to specify units:¶

q.mm, q.m, q.km

(1 mm, 1 m, 1 km)

2 * q.mm**2
\[2 $\text{mm}^{2}\]
v = 30 * q.m / q.s
v.to('km/h')
\[108.00 $\frac{\text{km}}{\text{h}}\]
p = 20*q.dBm + 3*q.dB

p, p.W

(23 dBm, 0.20 W)
